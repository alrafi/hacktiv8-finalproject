### Configure

Get the repo

```sh
git clone https://github.com/alrafi/hacktiv8-finalproject.git
cd hacktiv8-finalproject
npm install
npm start
```

Add `.env` file to your root directory and add the following

```js
REACT_APP_API_KEY = [YOUR_OMDB_APIKEY];
```
